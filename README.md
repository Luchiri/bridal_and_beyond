# Bridal and Beyond

Bridal and Beyond is a Django-based web application that allows users to create and manage boards for wedding planning, including categories such as Bridal, Groom, Accessories, and Decor. Users can upload images, organize inspirations, and explore other users' boards for ideas. 

## Features
- User authentication and profile management
- Create, view, and manage boards
- Upload images and set cover images
- Categorize boards into Bridal, Groom, Accessories, or Decor
- Responsive and user-friendly interface

## Technologies Used
- Python 3
- Django 5
- HTML, CSS, JavaScript
- SQLite (default database)
